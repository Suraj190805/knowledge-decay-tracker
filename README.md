# 📘 Knowledge Decay Tracker

A full-stack AI-powered learning platform that helps students retain knowledge effectively by transforming study materials into quizzes, summaries, and analytics.

---

## 🚀 Features

- 📂 Upload study materials (PDFs, images)
- 🧠 AI-generated quizzes from content
- 📝 Automatic summary generation
- 📊 Learning analytics dashboard
- 👤 User profile with editable details
- 🔗 Social links integration with hover interaction
- 💾 Persistent data using local storage & backend

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- CSS (Custom styling)

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI / Processing
- Groq API (LLM)
- PDF-Parse
- Multer (File Upload)

🧠 How It Works
	1.	Upload study material (PDF/image)
	2.	Backend extracts text using PDF parser
	3.	AI generates:
	•	Quiz questions
	•	Summaries
	4.	User can:
	•	Attempt quizzes
	•	Track progress
	•	Review summaries
